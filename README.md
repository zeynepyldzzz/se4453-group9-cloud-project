# Midterm Project Backend

## Tech Stack
- Python
- Flask
- PostgreSQL

## Endpoints
- GET /
- GET /hello

## Required Environment Variables
- DB_HOST
- DB_PORT
- DB_NAME
- DB_USER
- DB_PASSWORD

## Run Locally
1. Create a virtual environment
2. Install dependencies with `pip install -r requirements.txt`
3. Create a `.env` file
4. Run `python app.py`
